# VigilumSP

## ⚙️ Configuração do Ambiente - Coleta de Dados

### Criando e Ativando o Ambiente Virtual

**Windows:**
```bash
python -m venv 
.venv\Scripts\activate
```

**Mac/Linux:**
```bash
python3 -m venv
source .venv/bin/activate
```

### Instalação das Bibliotecas

```bash
pip install -r requirements.txt
```
